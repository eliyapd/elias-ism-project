How do computers do math?

Prior to discussing quantum computing, let's first cover how a classical computer (like the one you might be using right now!) performs calculations.

Before a computer can do math like we do, it must first convert decimal numbers to binary numbers. This is because a computer's hardware is made of transistors, which have two different states: on and off. The number '1' represents current flow within the computer, and the number '0' represents the absence of these signals.
